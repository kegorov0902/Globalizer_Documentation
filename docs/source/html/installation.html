Поддержка платформ

Система Globalizer поддерживает установку на следующие платформы:

  • Linux (Ubuntu/Debian)
  • Windows
  • macOS
  • Docker
  • Требования к системе
     Минимальные и рекомендуемые системные требования
  • Установка зависимостей
     Установка необходимых библиотек и инструментов
  • Клонирование репозитория
     Для начала клонируйте репозиторий с исходным кодом на свой компьютер:
     git clone https://github.com/OptimLLab/Globalizer.git
     Далее перейдите в папку с кодом из репозитория и создайте в ней папку "build":
     cd Globalizer/
     mkdir build
  • Сборка
     Перейдите в папку "build":
     cd build
     Выполните следующие команды:
     cmake ..
     cmake --build ../build --config Release
  • Проверка установки
     Тестирование работоспособности системы
Предварительные требования:
  • Операционная система: Linux, Windows или macOS
  • Компилятор C++ с поддержкой стандарта C++17
  • CMake версии 3.15 или выше
  • Git для клонирования репозитория
  • Дополнительно: CUDA для GPU-ускорения (опционально)
