Подробное описание

Базовый класс-интерфейс, от которого наследуются классы, описывающие задачи оптимизации использующие для вычислений GPU.

В классе IGPUProblem описаны прототипы методов, которые должны быть реализованы в подключамых модулях с решением на GPU.
