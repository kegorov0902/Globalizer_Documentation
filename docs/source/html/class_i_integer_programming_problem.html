Подробное описание

Базовый класс-интерфейс, от которого наследуются классы, описывающие задачи оптимизации с дискретными параметрами.

В классе #TIProblem описаны прототипы методов, которые должны быть реализованы в подключамых модулях в которых задачи используют дискретный параметр.

Для дискретных параметра задаются все допустимые значения. Дискретные параметры являются последними в векторе параметров y.
