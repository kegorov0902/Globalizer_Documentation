Подробное описание

The Rastrigin function is given by the formula:
   :math:`f(y)=(\sum_{i=1}^{N}[x_{i}^{2}-10*cos(2\pi x_{i})])`,
   where :math:`x\in [-2.2, 1.8], N` – problem dimensionality
