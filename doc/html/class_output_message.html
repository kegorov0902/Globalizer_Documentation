Инициализация систем вывода

Аргументы
[in]isMPIInit- Проинициализирован ли MPI, требуется для корректной печати в файл в многопроцессорном режиме
[in]mLogFileName- имя (префикс) лог файла
[in]processCount- число процессов
[in]processNumber- номер текущего процесса
[in]isPrintToFile- печатать ли в файл все сообщения в debug, в release не печатается
[in]errorsName- имена (расшифровка) ошибок
[in]errorsCode- коды ошибок
[in]errorsCount- число ошибок последние три параметра инициализируются по умолчанию в методе SetDefaultErrors
